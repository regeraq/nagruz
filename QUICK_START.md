# ⚡ Быстрый Запуск Сервера

## Если Node.js только что установлен

После установки Node.js **обязательно перезапустите терминал/IDE**, чтобы переменные окружения обновились.

## Проверка установки

Откройте **новый** терминал PowerShell и выполните:

```powershell
node --version
npm --version
```

Если команды работают, переходите к следующему шагу.

## Запуск проекта

1. **Перейдите в папку проекта:**
```powershell
cd "C:\Users\Andrey\Downloads\HelloWhoAreYou-1 (1)\HelloWhoAreYou-1"
```

2. **Установите зависимости (первый раз):**
```powershell
npm install
```

Это может занять 2-5 минут при первой установке.

3. **Запустите сервер разработки:**
```powershell
$env:NODE_ENV="development"
npm run dev
```

4. **Откройте в браузере:**
```
http://localhost:5000
```

## Если Node.js не найден

### Вариант 1: Перезапуск терминала
- Закройте текущий терминал/IDE
- Откройте новый терминал
- Попробуйте снова

### Вариант 2: Проверка установки
1. Откройте "Переменные среды" в Windows
2. Проверьте, что `C:\Program Files\nodejs` есть в PATH
3. Если нет - добавьте вручную

### Вариант 3: Переустановка Node.js
1. Скачайте с [nodejs.org](https://nodejs.org/)
2. Выберите LTS версию
3. При установке убедитесь, что выбрана опция "Add to PATH"

## Что вы увидите после запуска

✅ Сервер запустится и покажет:
```
serving on port 5000 (development)
```

✅ В браузере откроется сайт с:
- Улучшенной безопасностью (rate limiting)
- Кэшированием (быстрые ответы API)
- Защитой от XSS
- Исправленными багами

## Остановка сервера

Нажмите `Ctrl+C` в терминале.

## Проблемы?

Если возникают ошибки:
1. Убедитесь, что порт 5000 свободен
2. Проверьте, что все зависимости установлены: `npm install`
3. Проверьте версию Node.js: должна быть 18+
